# 美土里農園 収穫・在庫管理システム

## 概要

このシステムでは、美土里農園で収穫された在庫を管理するシステム

## システム要件

- このサイトは html, css, java を用いた静的サイトとし、GitHub Pagesにてホスティングする。
- main ブランチを対象にホスティングしているため、main ブランチへの push = release と判断する。

## 詳細設計

- 管理する在庫はにんじん、さつまいも(べにはるか、シルクスイート)
  - 植えた日付、経過日数を表示する
- 農地のマップと、当該マップへ植えた食物を入力可能とする
  - マップサイズは以下
    □□□□□□
    □□□□□□
    □□
    □□
- 各食物の収穫・出荷・在庫をグラフ化する
  - 在庫を棒グラフ、収穫と出荷を折れ線グラフ
- 農地マップの区画ごとに土地の状態を管理する
  - 肥料の種類（自由入力）
  - 最終施肥日
  - 土のpH
- 天候を都度記録できる（日付・区画・晴れ/曇り/雨）
  - 農地マップ画面内に天候記録セクションを設ける
- 振り返り画面を設ける
  - 収穫記録に対して、その区画の肥料・pH・収穫日の天候・生育期間中の天候を紐づけて表示する
- 農作業タスク管理画面を設ける
  - タスクは農地マップの区画に紐づける
  - 定型タスク：水やり、追肥、除草、土寄せ、収穫
  - 自由入力タスクも可能
  - 期日を設定できる
  - ブラウザ通知（Web Notifications API）で期日当日・超過を通知する
  - タスクの完了チェック機能を設ける
