# 美土里農園 収穫・在庫管理システム

## 概要
美土里農園で収穫された在庫を管理するシステム

## ユーザー
- 入力担当：農園スタッフ（67歳・スマホから入力）
- 管理・閲覧：農園管理者（PCから閲覧メイン）

## システム要件
- html, css, javascript を用いた静的サイト
- GitHub Pagesにてホスティング
- mainブランチへのpush = release と判断する

## 機能一覧

### 1. 農地マップ管理（新規追加）
- 農地はピストル型、以下の16区画で構成される
  - 1列目：6区画
  - 2列目：6区画
  - 3列目：2区画（右寄り）
  - 4列目：2区画（右寄り）
- 各区画の中に初期状態として畝が4つ、畝と畝の間には通路（畝間）がある
  - 畝は追加・削除が可能
  - 畝の向き（縦・横）を切り替え可能
- 各区画（マップ上のセル）に視覚的な状態を表示する
  - 畝の状態：🌱（栽培中）、🥕/🍠（収穫期超過）、空白（未栽培）
  - 畝間（草むしり）の状態：🌿（期限超過）、細線（完了・予定内）
- 各畝に以下を記録・管理できる
  - 作物の種類・品種（作物マスタから選択）
  - 播種・定植日
  - 収穫予定日
  - 使用した有機肥料（肥料マスタから複数選択可）
  - この畝の栽培を終了（履歴へ記録）
- 畝間ごとに以下を記録・管理できる
  - 次回草むしり予定日
  - 草むしり完了（履歴へ記録）
- 区画ごとの「履歴を見る」画面
  - 過去の栽培完了履歴、草むしり完了履歴を年ごとに表示

### 2. 在庫・農地・タスク・マスタ管理機能
- 管理作物・肥料はマスタ管理画面（⚙️ マスタ管理タブ）から追加・削除できる
  - 年によって作物や肥料が変わることを考慮した設計とする
  - 初期登録作物：
    - にんじん（夏まき・冬まき）
    - さつまいも（紅はるか・シルクスイート・紅あずま）
- 植えた日付・経過日数を表示
- 収穫した数量・重量を記録
- 出荷・販売の記録
- 廃棄・ロスの記録
- 価格・売上の記録
- グラフ表示
  - 在庫：棒グラフ
  - 収穫・出荷：折れ線グラフ
- 農地マップの区画ごとに土地の状態を管理する
  - 肥料の種類（自由入力）
  - 最終施肥日
  - 土のpH
- ダッシュボード（今日のタスク）機能
  - アプリ起動時に、全区画の収穫・草むしり状態を自動走査しアラートを表示
  - 収穫期限超過（🥕🍠）、収穫間近（3日以内）
  - 草むしり期限超過（🌿）、草むしり間近（3日以内）
- 天候を都度記録できる（日付・区画・晴れ/曇り/雨）
  - 農地マップ画面内に天候記録セクションを設ける
- 振り返り画面を設ける
  - 収穫記録に対して、その区画の肥料・pH・収穫日の天候・生育期間中の天候を紐づけて表示する
- 農作業タスク管理画面を設ける
  - タスクは農地マップの区画に紐づける
  - 定型タスク：水やり、追肥、除草、土寄せ、収穫
  - 自由入力タスクも可能
  - 期日を設定できる
  - ブラウザ通知（Web Notifications API）で期日当日・超過を通知する
  - タスクの完了チェック機能を設ける

### 3. UI要件
- スマホファーストのデザイン
- 文字・ボタンは大きめ
- 操作ステップは最小限に
- 難しい言葉を使わない
