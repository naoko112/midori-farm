# 美土里農園 収穫・在庫管理システム

## 概要
美土里農園で収穫された在庫を管理するシステム

## ユーザー
- 入力担当：農園スタッフ（67歳・スマホから入力）
- 管理・閲覧：農園管理者（PCから閲覧メイン）

## システム要件
- html, css, javascript を用いた静的サイト
- GitHub Pagesにてホスティング
- mainブランチへのpush = release と判断する

## 機能一覧

### 1. 農地マップ管理（新規追加）
- 農地はピストル型、以下の16区画で構成される
  - 1列目：6区画
  - 2列目：6区画
  - 3列目：2区画（右寄り）
  - 4列目：2区画（右寄り）
- 各区画の中に畝が4つ（畝1〜4）あり、畝と畝の間は通路（草刈り必要）
- 各畝に以下を記録できる
  - 作物の種類・品種
  - 播種・定植日
  - 収穫予定日
  - 農薬・肥料の使用記録

### 2. 在庫・農地・タスク管理機能
- 管理作物はマスタで管理し、管理画面から追加・編集・削除できる
  - 年によって作物が変わることを考慮した設計とする
  - 初期登録作物：
    - にんじん（夏まき・冬まき）
    - さつまいも（紅はるか・シルクスイート・紅あずま）
- 植えた日付・経過日数を表示
- 収穫した数量・重量を記録
- 出荷・販売の記録
- 廃棄・ロスの記録
- 価格・売上の記録
- グラフ表示
  - 在庫：棒グラフ
  - 収穫・出荷：折れ線グラフ
- 農地マップの区画ごとに土地の状態を管理する
  - 肥料の種類（自由入力）
  - 最終施肥日
  - 土のpH
- 天候を都度記録できる（日付・区画・晴れ/曇り/雨）
  - 農地マップ画面内に天候記録セクションを設ける
- 振り返り画面を設ける
  - 収穫記録に対して、その区画の肥料・pH・収穫日の天候・生育期間中の天候を紐づけて表示する
- 農作業タスク管理画面を設ける
  - タスクは農地マップの区画に紐づける
  - 定型タスク：水やり、追肥、除草、土寄せ、収穫
  - 自由入力タスクも可能
  - 期日を設定できる
  - ブラウザ通知（Web Notifications API）で期日当日・超過を通知する
  - タスクの完了チェック機能を設ける

### 3. UI要件
- スマホファーストのデザイン
- 文字・ボタンは大きめ
- 操作ステップは最小限に
- 難しい言葉を使わない
