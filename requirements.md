# 美土里農園 収穫・在庫管理システム

## 概要

このシステムでは、美土里農園で収穫された在庫を管理するシステム

## システム要件

- このサイトは html, css, java を用いた静的サイトとし、GitHub Pagesにてホスティングする。
- main ブランチを対象にホスティングしているため、main ブランチへの push = release と判断する。

## 詳細設計

- 管理する在庫はにんじん、さつまいも(べにはるか、シルクスイート)
  - 植えた日付、経過日数を表示する
- 農地のマップと、当該マップへ植えた食物を入力可能とする
  - マップサイズは以下
    □□□□□□
    □□□□□□
    □□
    □□
- 各食物の収穫・出荷・在庫をグラフ化する
  - 在庫を棒グラフ、収穫と出荷を折れ線グラフ
