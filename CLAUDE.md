# CLAUDE.md

## 要件
- このシステムの要件は requirements.md を参照すること

## 技術方針
- html, css, javascript のみを使用した静的サイト
- フレームワーク・ライブラリは極力使わない
- データはlocalStorageに保存する
- ファイルは index.html 一本にまとめる

## デザイン方針
- スマホファーストのデザイン
- 文字・ボタンは大きめ
- 操作ステップは最小限に
- 難しい言葉を使わない
- カラーは既存のCSS変数を引き継ぐこと

## 制約事項
- GitHubのmainブランチへのpush = releaseである
  - releaseの前にかならず確認すること
- publicリポジトリであるため、個人情報やクレデンシャル情報が入力された場合はreleaseを中止する
- わからない部分は勝手に補完せず、問い合わせを行う
- 会話の中で確定した要件は requirements.md に記載する
  - 新しく出てきた要件によって過去の要件に修正が生じた場合は問い合わせを実施するs
